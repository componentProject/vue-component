# description

这是一个vue的组件库,在实现组件的同时,探索 vite & vitepress & storybook 别样的魅力

# 下载依赖并启动
```shell
pnpm install
pnpm dev
```

# 下载依赖并启动storybook
```shell
pnpm install
pnpm storybook:dev
```
